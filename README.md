# pi-messaging

Exploring a generalized mechanism for 8 bit messaging between a TI-99/4A and
a Raspberry PI 3

## Who: 

* ElectricLab
* Jedimatt42

## File Types

Schematic files will be in KiKad format.
Initial code will be in gcc for the TI and python for the Raspberry PI.
